# 🎯 Smart Object Tracking System

A real-time object tracking project built with **Python** and **OpenCV** using the **CSRT tracking algorithm**.

## ✨ Features

- Real-time object tracking
- CSRT tracking algorithm
- Bounding box and motion trail
- FPS, position, and object size display
- Screenshot capture
- Video recording with `REC` indicator
- Reset and select a new object without restarting

## 🛠️ Technologies

- Python
- OpenCV
- Anaconda
- PyCharm

## ⚙️ Installation

```bash
conda create -n opencv-tracking python=3.11
conda activate opencv-tracking
pip install -r requirements.txt
```

## ▶️ Run
![Tracking Example](TrackingExample.png)
```bash
python object_tracker.py
```

## 🎮 Controls

| Key | Action |
|---|---|
| `R` | Select / reset object |
| `S` | Save screenshot |
| `V` | Start / stop recording |
| `Q` | Quit |
